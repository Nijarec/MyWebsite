Proyecto Modular Universidad de Guadalajara - GESTOR DE PROYECTO

Realicé el proyecto final de mi carrera universitaria como ingeniero en informática, el cual llevó por nombre “GearWater”. Este proyecto consistió en el desarrollo de una aplicación para teléfonos móviles y otros dispositivos, enfocada en calcular la cantidad de agua necesaria para irrigar determinado cultivo, considerando el número de hectáreas y la ubicación geográfica proporcionada por el usuario. El cálculo se realizó mediante operaciones complejas que determinaban la cantidad de agua requerida, basándose en los datos ingresados por el usuario y en información obtenida por la propia aplicación, como el estado del clima, las horas de luz solar, la evaporación del agua, entre otros factores. El proyecto se completó de manera exitosa, cumpliendo con sus objetivos y obteniendo la aprobación del comité universitario, satisfaciendo plenamente sus expectativas. Todo esto lo hice usando Java dentro de Android Studio.

Desarrollé una aplicación de escritorio llamada “ECS: Encrypter Control Security” capaz de encriptar y desencriptar textos utilizando una clave de acceso personalizada creada por el usuario.
